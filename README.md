# 📊 Dashboard de Análise de Vendas - Power BI

Projeto desenvolvido como desafio prático da DIO com foco na construção de um relatório interativo e otimizado para experiência do usuário (UX), utilizando o Power BI.

---

## 🎯 Objetivo

Modificar um relatório inicial, aplicando boas práticas de:

- Posicionamento dos elementos
- Contraste visual
- Proporção e organização
- Segmentação dos dados
- Experiência do usuário (UX)

Além disso, foram implementados recursos de navegação e interatividade para tornar a análise mais intuitiva.

---

## 🚀 O que foi desenvolvido

✔️ Dashboard com **3 páginas principais**:
- **Home** → Página inicial com navegação  
- **Relatório** → Visão geral das vendas  
- **Detalhes / TOP N** → Análise aprofundada  

✔️ Implementação de:
- Botões de navegação entre páginas  
- Menu lateral interativo  
- Destaque visual em botões (hover/seleção)  
- Segmentação por data  
- Layout organizado e intuitivo  

✔️ Uso de:
- Gráficos de barras e colunas  
- Treemap  
- Indicadores (KPIs)  
- Filtros interativos  

---

## 📊 Principais análises

O dashboard permite analisar:

- Vendas por período  
- Vendas por segmento  
- Vendas por produto  
- Distribuição geográfica (países)  
- TOP N produtos mais relevantes  

---

## 💡 Insights obtidos

- O produto **Paseo** lidera o volume de vendas no TOP 3.  
- **Estados Unidos e Canadá** concentram os maiores resultados.  
- A análise TOP N facilita a identificação rápida dos produtos mais relevantes.  
- Existe concentração de vendas em poucos produtos, indicando possível dependência comercial.  

---

## 🎨 Experiência do Usuário (UX)

O relatório foi projetado com foco na usabilidade:

- Layout limpo e organizado  
- Hierarquia visual clara  
- Uso estratégico de cores (roxo + amarelo)  
- Navegação intuitiva entre páginas  
- Elementos visuais para guiar a análise  

---

## 🛠️ Ferramentas utilizadas

- Power BI  
- Modelagem de dados  
- DAX (medidas básicas)  
- Técnicas de Data Visualization  

---

## 📌 Conclusão

Este projeto demonstra a importância de unir análise de dados com design e experiência do usuário, tornando dashboards mais eficientes, intuitivos e estratégicos para tomada de decisão.
